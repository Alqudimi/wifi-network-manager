# الواجهة الأمامية (Frontend)

هذا المجلد يحتوي على الكود الخاص بالواجهة الأمامية لمشروع إدارة شبكات Wi-Fi.

## التقنيات المستخدمة

- **الإطار**: React (Vite)
- **اللغة**: JavaScript/TypeScript
- **التصميم**: RTL (عربي/إنجليزي) متجاوب

## الإعداد والتشغيل

1.  **المتطلبات المسبقة**:
    - Node.js (LTS)
    - npm أو yarn

2.  **الاستنساخ والتثبيت**:
    ```bash
    git clone <رابط المستودع>
    cd wifi-network-manager/frontend
    npm install   # أو yarn install
    ```

3.  **إعداد المتغيرات البيئية**:
    قم بإنشاء ملف `.env` في هذا المجلد بناءً على `.env.example` واملأ المتغيرات المطلوبة (مثل `VITE_API_BASE_URL`).

4.  **تشغيل خادم التطوير**:
    ```bash
    npm run dev   # أو yarn dev
    ```
    سيتم تشغيل التطبيق على `http://localhost:5173` (أو منفذ آخر).

## البناء (Build)

لبناء التطبيق للإنتاج:

```bash
npm run build   # أو yarn build
```

سيتم إنشاء ملفات البناء في مجلد `dist/`.

## الاختبارات

لتشغيل اختبارات الواجهة الأمامية:

```bash
npm test   # أو yarn test
```

## النشر (Deployment)

يتم توفير `Dockerfile` لسهولة النشر.
